Alya Temp File Transfer

Temporary and encrypted transfer everywhere!!

Usage

You can easily use this app via Browsers.
Upload via cURL:
curl --upload-file ./filename.txt url (e.g. curl --upload-file ./hello.txt https://alya-temp-file.herokuapp.com/api/hello.txt)
Retrieve via cURL:
curl url (e.g. curl https://alya-temp-file.herokuapp.com/api/f-sadEsse/hello.txt)
